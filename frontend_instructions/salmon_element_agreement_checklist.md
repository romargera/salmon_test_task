# Salmon Direct Contrast: Element Agreement Checklist

Используем этот список как обязательный preflight перед генерацией презентации, чтобы слайды были визуально консистентны.

## 1) Базовый каркас
- Формат: 16:9 (`13.333 x 7.5`).
- Поля: `0.84` слева/справа, `0.64` сверху/снизу.
- Фон всех слайдов: `#FFFFFF`.

## 2) Типографика
- Heading: `Arial Rounded MT Bold`.
- Body: `Arial`.
- Заголовки только `#141414`.
- Вторичный текст только `#555555`.

## 3) Цветовая дисциплина
- Красный акцент только `#F45B5F`.
- Основной текст в плашках: чёрный/графитовый.
- Не использовать white text на красной заливке для основных текстовых блоков.

## 4) Компоненты
- CTA: белая кнопка, красный бордер, красный текст.
- Card: белая карточка + тонкий бордер `#E3DFD9`.
- Icons: белый круг + красный контур + красный символ.
- Tables: белая база, ключевые значения красным текстом.
- Charts: один акцентный красный ряд/серия, остальные нейтральные.

## 5) Ограничения контента
- Заголовок: до 8 слов.
- Буллет: до 12 слов.
- Буллетов на слайде: до 5.
- Диаграмма: до 6 категорий.
- Таблица: до 5 колонок.

## 6) Техническая интеграция
- В metadata фиксировать `style_preset_id = salmon_direct_contrast_v1`.
- Применять токены из `salmon_direct_contrast_style_v1.json` без ручных override.
- Все исключения фиксировать отдельно в notes к слайду.
