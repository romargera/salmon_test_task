# Salmon Presentation Style Guide (Direct Contrast v1)

## 1) Зафиксированный стиль
- База: светлый фон (`#FFFFFF`) и светлые поверхности.
- Текст: в первую очередь чёрный/графитовый, без белого текста на красном фоне.
- Акцент: фирменный coral (`#F45B5F`) как единственный активный акцент.
- Принцип: `white + black + coral`, минимум вторичных цветов.

## 2) Design tokens
```json
{
  "style_preset_id": "salmon_direct_contrast_v1",
  "palette": {
    "bg_base": "#FFFFFF",
    "surface": "#FFFFFF",
    "surface_soft": "#F8F6F3",
    "stroke": "#E3DFD9",
    "text_primary": "#141414",
    "text_secondary": "#555555",
    "text_muted": "#777777",
    "brand_coral": "#F45B5F",
    "chart_neutral": "#C9D0D7"
  },
  "fonts": {
    "heading": "Arial Rounded MT Bold",

    "h1": 54,

  "shape": {
    "card_radius": 0.14,

  },
  "spacing": {
    "outer_margin": 0.84,
    "grid_gap": 0.28,
    "section_gap": 0.44
  }
}
```

## 3) Глобальные правила консистентности
- Micro-label в левом верхнем углу каждого слайда, цвет `brand_coral`.
- CTA: белая кнопка, красная обводка, красный текст.
- Карточки: светлые (`surface`) с тонкой обводкой `stroke`.
- Заголовки всегда `text_primary`, вспомогательный текст — `text_secondary`.
- Не использовать плашки с красной заливкой под длинный текст.

## 4) Внешний вид элементов (согласовано)
- `title`: крупный заголовок слева, справа инфо-карточка с CTA.
- `bullet_list`: 3-5 пунктов, маркеры красные контурные, текст чёрный.
- `two_column`: обе колонки белые с тонкой обводкой; правую колонку акцентировать красным заголовком.
- `table`: белая таблица, чёрный текст, ключевая колонка/значение красным текстом.
- `chart`: нейтральные столбцы серые, основной бар/линия — coral.
- `quote`: белый блок, слева тонкая красная вертикальная линия.
- `closing`: один крупный чёрный тезис + красный micro-label + контакт `text_muted`.

## 5) Контентные ограничения
- Заголовок: до 8 слов.
- Подзаголовок: до 18 слов.
- Буллеты: до 5 пунктов, каждый до 12 слов.
- Таблицы: максимум 5 колонок на слайд.
- Графики: максимум 6 категорий на один chart.

## 6) Для вашего pipeline
- Этот пресет нужно передавать в `metadata.style_preset_id = "salmon_direct_contrast_v1"`.
- Если используете `python-pptx-fix`, храните в JSON явные `radius`, `padding`, `line_height`, `stroke_width`.
- При генерации каждого слайда применять одни и те же токены из `palette/fonts/shape/spacing`.
