# группа в revolut - туда можно пригласить участника без революта? какой критерий входа, чтобы

1. написать сообщение в группе револют
2. занести расход
3. сделать перевод

Да, **пользователя без Revolut можно пригласить в группу** (Group Bills). Это официально поддерживаемая функция — такой участник получает доступ через сайт **groupbills.revolut.com**, а не через приложение.[^1_1]

## Как присоединяется участник без Revolut

Не-Revolut пользователь создаёт базовый профиль на groupbills.revolut.com, подтверждая только свой **email** (аутентификация — одноразовый код на почту). Никакого аккаунта Revolut, верификации личности или карты не нужно.[^1_1]

## Что можно делать — по ролям

| Действие | Пользователь Revolut | Без Revolut |
| :-- | :-- | :-- |
| **Написать в чате группы** | ✅ (в приложении) | ❌ (нет доступа к чату в приложении) |
| **Добавить/занести расход** | ✅ | ✅ (через сайт groupbills.revolut.com) |
| **Отметить долг как оплаченный** | ✅ | ✅ |
| **Перевод через Revolut (P2P)** | ✅ | ❌ |
| **Погасить долг по платёжной ссылке** | ✅ | ✅ (вводит данные карты/банка на сайте) |
| **Сгенерировать платёжную ссылку** | ✅ | ❌ |

## Ключевые ограничения для участников без Revolut

- **Сообщения в чате группы** — доступны только через приложение Revolut, т.е. без аккаунта написать в групповой чат **нельзя**[^1_1]
- **Занести расход** — можно через веб-версию groupbills.revolut.com[^1_1]
- **Перевод** — самостоятельно инициировать перевод через Revolut нельзя; единственный способ погасить долг — получить **платёжную ссылку от участника с Revolut** и оплатить картой или банковским переводом[^1_1]

Важно: Group Bills — это **инструмент учёта расходов**, не платёжный сервис. Реальное движение денег происходит за его пределами — сама платформа только фиксирует, кто кому сколько должен.[^1_1]
