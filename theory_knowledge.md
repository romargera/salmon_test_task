### Network effect

Network effect отвечает на вопрос: растет ли ценность продукта для пользователя по мере роста числа других участников сети или другой группы пользователей.

* Прямой эффект: чем больше пользователей в той же группе, тем выше ценность продукта для каждого из них, например мессенджер или социальная сеть.
* Непрямой эффект: одна группа пользователей делает продукт ценнее для другой группы пользователей, например sellers для buyers, drivers для riders, creators для consumers.

### Если будешь использовать эти термины, то кратко объясни значение в скобках

* Remittances — переводы денег в другую страну
(работаешь за границей → отправляешь деньги домой)
* Escrow — деньги временно хранит третья сторона до выполнения условий
(маркетплейс держит оплату пока ты не получишь товар)
* Overdraft — можно уйти в минус по счёту
(на карте 0 €, заплатил → стало −50 €)
* Hedging — защита от финансовых рисков / колебаний курса
(фиксируешь курс заранее)
* Lending — выдача кредита
(берёшь деньги → возвращаешь с процентами)
* AUM-driven — компания зарабатывает % от суммы денег под управлением
(инвест-сервис берёт комиссию от портфеля)
* Gig payouts — быстрые выплаты фрилансерам / курьерам
(сделал заказ → сразу получил деньги)
* Factoring — бизнес получает деньги за будущий платёж сразу
(клиент заплатит через 60 дней → сервис платит тебе сейчас меньше)
* Off-ramp. Когда пользователь продаёт крипту и получает обычные деньги.
* On-ramp. Когда пользователь покупает крипту за обычные деньги.

### Про валюты

На Филиппинах официальная валюта — филиппинское песо (PHP, символ ₱).
Валюта страны
Филиппинское песо является официальной валютой Филиппин и используется для внутренних расчетов в стране.
Топ-2 валюта по доле
Если вы имеете в виду валютную структуру внешнего долга Филиппин, то на первом месте находится доллар США с долей 70.9%, а на втором — филиппинское песо с долей 11.5%.
Следом идет японская иена с долей 7.6%, а евро занимает 6.6%.
